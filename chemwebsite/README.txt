To run the website: 

first install python on your system from https://www.python.org/downloads/

Add python to your system environment variables so that you can run pip from any directory through cmd

open the folder chemwebsite in your cmd using the cd command

run the command - pip install -r requirements.txt

run the command - python app.py 

then the website will run locally on your system at localhost:5000

to add pdf files just drop them in the chemwebsite folder it will appear on the html page for download
